## 📊 数据库结构

### 1. user_info（个人信息表）
- id：主键
- avatar：头像
- name：姓名
- intro：简介
- resume：简历（JSON）
- contact：联系方式（JSON）
- update_time：更新时间

### 2. carousel（轮播图表）
- id：主键
- image：图片
- sort：排序

### 3. works（作品表）
- id：主键
- title：标题
- cover_image：封面图
- desc：描述

- create_time：创建时间

### 4. hobbies（爱好表）
- id：主键
- name：名称
- icon：图标类名
- desc：描述

## 🎨 自定义配置

### 修改主题色
编辑 `frontend/tailwind.config.js`：
```javascript
colors: {
  'sakura-pink': '#FFB7C5',    // 樱花粉
  'sakura-light': '#FFF0F5',   // 浅粉
  'sakura-dark': '#FF69B4',    // 深粉
}
```

```javascript
const petalCount = 20  // 花瓣数量
```

```javascript
autoPlayTimer = setInterval(next, 5000)  // 5秒切换
```

## 🐛 常见问题

### Q: 图片不显示？
A: 需要在后台上传实际图片，初始数据使用的是占位符。

### Q: 跨域错误？
A: 检查 `settings.py` 中的 `CORS_ALLOWED_ORIGINS` 配置。

### Q: 端口被占用？
A: 修改端口号或关闭占用端口的程序。

### Q: 动画卡顿？
A: 减少花瓣数量或关闭其他占用资源的程序。
